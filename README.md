## Descrição do projeto
* A GameStore é um projeto de página de uma loja vitual de jogos de vídeo game. Nela é possível encontrar <br>
jogos de diferentes gêneros, desde ação, aventura, esporte e muito mais. O usuário pode optar por ver o catálogo <br>
completo de jogos disponíveis ou se preferir, pode optar por aplicar um filtro e ver os jogos disponíveis por gênero.

<p align="center"> <img src="https://imgur.com/bNsS4OH" alt="Catálogo da GameStore"> </p>

* A página também conta com a aba "Iniciar sessão" onde o usuário é enviado para outra página onde o mesmo pode<br>
realizar o cadastro na loja através da validação de formulário.
